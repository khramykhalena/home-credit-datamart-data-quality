# Описание проекта

## Название
Подготовка витрины данных и анализ качества данных по кредитным заявкам

## Бизнес-контекст
У компании есть исторические данные по заявкам клиентов, внешней кредитной истории и финансовым показателям.
Для аналитики и BI сырые таблицы неудобны: они содержат пропуски, аномалии и связи формата `1:N`.
Нужна единая витрина, где каждая строка соответствует текущей заявке клиента, а дополнительные данные по кредитной истории уже агрегированы.

## Цель
Собрать аналитическую витрину, пригодную для:
- BI-дашборда;
- сегментации клиентов;
- последующего системного анализа процесса заявки;
- построения скоринговых и исследовательских моделей.

## Ключевой результат
Финальная витрина уровня `1 строка = 1 заявка` c базовыми признаками заявки и агрегатами по внешней кредитной истории.
